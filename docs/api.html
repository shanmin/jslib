$js().add

定义

$js().add(selector) : JSLib

$js().add(HtmlString) : JSLib

$js().add(HtmlElement) : JSLib

参数

selector
[可选]指定选择器字符串。
[optional]A string containing one or more selectors to match.
HtmlString [可选]指定的HTML字符串。
HtmlElement [可选]指定HTML节点。

说明

向当前匹配节点列表中添加新的节点。

示例

HTML 
<div id="a1">
	<div id="a2">a2 text</div>
	<div id="a3">a3 text</div>
</div>
JavaScript 
$js('#a2').add('#a3').addClass('newclass');

当执行$js('#a2')时，此对象会匹配a2节点。再执行add后，此对象会增加a3节点到匹配列表中。

执行后原HTML信息变为 
<div id="a1">
	<div id="a2" class="newclass">a2 text</div>
	<div id="a3" class="newclass">a3 text</div>
</div>

版本

本方法自0.0.0版本开始支持。

$js.ajax

定义

$js.ajax(opts) : XHR

参数

opts [必须]指定的访问参数。
opts.headers [可选]访问头部数据。
opts.url [可选]访问的文件路径。如果未传递此属性，则默认为当前页面。
opts.method [可选]访问方式。可选参数：GET、POST、PUT，默认：GET。
opts.type [可选]method参数的别名。如果同时设置了method和type则以method为准。
opts.dataType [可选]返回数据类型，可选参数：json。
设置“json”类型，则回调函数success中的data数据类型会变为解析json后的对象，如果解析失败则会调用error回调函数。
设置“arraybuffer”类型，则回调中的data为ArrayBuffer类型。
opts.post [可选]发送的POST信息。如果传递的是一个Object类型，则会自动转换为字符串。
opts.interval [可选]周期性请求的时间间隔，单位：ms。注意：这个时间间隔是按照上次访问并处理后开始计时的。
opts.success=function()|String [可选]请求成功后的回调的函数或后续操作符。
回调函数定义：function(data,status,xhr)，data为返回的数据，status为HTTP请求状态码，xhr为访问组件。
如果设置字符串"reload"，则成功访问后会自动重新加载当前页。
opts.error [可选]请求失败后的回调的函数。函数定义function(xhr,status,ex)，xhr为访问组件，status为HTTP请求状态码，ex报错信息。
opts.async [可选]异步请求，默认：true。

说明

AJAX方式访问数据。

示例

JavaScript 
$js.ajax({url:'test.html'});

版本

本方法自0.1.0版本开始支持。

本方法自0.2.11版本开始支持success=reload参数。

			[可选]设置提醒信息关闭时间，单位：ms。此参数仅针对正常信息，出错信息不自动关闭。
		

	

	
说明

如果参数opts.success=reload，且返回数据ret[success]=true时默认重新加载当前页面；返回ret[success]=false时显示错误信息。

	
如果参数opts.success=reload2，且返回数据ret[success]=true时默认重新加载当前页面（reload2采用Refresh方式，可实现访问的页面刷新后显示到刷新前的位置）；返回ret[success]=false时显示错误信息。

	
如果参数opts.success=302，且返回数据ret[success]=true时页面转到opts.location页面；返回数据ret[success]=false时显示错误信息。

	
如果参数opts.success=302id，则如果当前页面URL中的参数id与返回data.id不一致时做302操作，否则不做任何操作。

	
如果未设置参数error，则出错的事后会自动调用参数success。

	
数据接收文件，在数据处理后需要返回JSON格式结果，标准最小格式，正常为：{success:true}，出错为：{success:false}

定义

参数

	

		

			value

	<script>
		window.frameElement.close();
		window.frameElement.close(true);
	</script>

		var win;
