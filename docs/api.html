$js().add

定义

$js().add(selector) : JSLib

$js().add(HtmlString) : JSLib

$js().add(HtmlElement) : JSLib

参数

selector [可选]指定选择器字符串。
HtmlString [可选]指定的HTML字符串。
HtmlElement [可选]指定HTML节点。

说明

向当前匹配节点列表中添加新的节点。

示例

HTML 
<div id="a1">
	<div id="a2">a2 text</div>
	<div id="a3">a3 text</div>
</div>
JavaScript 
$js('#a2').add('#a3').addClass('newclass');

当执行$js('#a2')时，此对象会匹配a2节点。再执行add后，此对象会增加a3节点到匹配列表中。

执行后原HTML信息变为 
<div id="a1">
	<div id="a2" class="newclass">a2 text</div>
	<div id="a3" class="newclass">a3 text</div>
</div>

版本

本方法自0.0.0版本开始支持。

$js.ajax

定义

$js.ajax(opts) : XHR

参数

opts [必须]指定的访问参数。
opts.url [可选]访问的文件路径。如果未传递此属性，则默认为当前页面。
opts.method [可选]访问方式。可选参数：GET、POST、PUT，默认：GET。
opts.type [可选]method参数的别名。如果同时设置了method和type则以method为准。
opts.dataType [可选]返回数据类型，可选参数：json。设置了json类型，则回调函数success中的data数据类型会变为解析json后的对象，如果解析失败则会调用error回调函数。
opts.success [可选]请求成功后的回调的函数。函数定义function(data,status,xhr)，data为返回的数据，status为HTTP请求状态码，xhr为访问组件。
opts.error [可选]请求失败后的回调的函数。函数定义function(xhr,status,ex)，xhr为访问组件，status为HTTP请求状态码，ex报错信息。
opts.async [可选]异步请求，默认：true。

说明

AJAX方式访问数据。

示例

JavaScript 
$js.ajax({url:'test.html'});

版本

本方法自0.1.0版本开始支持。

$js.regressionPolynomial

	
定义

	
$js.regressionPolynomial(data,order) : object

	
参数

			[必须]需要做回归计算的数据。

说明

	
对指定数据进行多项式回归。计算后返回一个固定对象：

	
	{
		data:[...],		//原始有效回归数据
		parameter:[...],	//多项式参数
		expresion:string	//显示使用的多项式字符串
		points:[[x,y],...],	//数据节点x对应回归后的数据。如果需要增加此展示点，可以调用calc或calcY方法进行添加

		calcY:function(y){},	//内部调用了$js.regressionCalcY，并且此计算点也会自动增加到points展示点列表中
	}

	
这个方法可以独立使用，也可以配合echarts使用。像points就是专门为echarts显示准备的。

$js.tree / $js().tree

	
定义

	
$js.tree(opts) : HtmlElement

	
$js().tree(opts) : HtmlElement

	
参数

	

		

			opts
			[必须]生成树的选项。
		

		

			opts.data
			[可选]树型结构数据。
		

		

			opts.dataurl
			[可选]树型结构从网络读取的URL。此参数与opts.data只能设置一个。此URL返回的数据格式参考opts.data说明。
		

		

			opts.def
			[可选]默认数据项,这个是与opts.data[].i比较得出的。
		

		

			opts.drag=false|true
			[可选]是否支持位置拖拽。默认值：false。
		

		

			opts.ondragfinish=function(e1,e2){}
			[可选]拖拽后节点位置e1操作节点，e2目标节点。
		

		

			opts.expand=false|true
			[可选]自动展开所有子节点。默认值：false。
		

		

			opts.root_expand=false|true
			[可选]自动展开根节点。默认值：false。
		

		

			opts.onclick
			[可选]节点点击时处理的回调函数。
		

		

			opts.onmousedown
			[可选]节点mousedown事件的回调函数。
		

		

			opts.parent
			[可选]生成树型结构的上层元素(HtmlElement)或元素ID(String)。注意，实例方法调用时会忽略此参数。
		

		

			opts.url
			[可选]节点默认链接,如果节点数据存在a，则忽略此参数。此参数支持自定义变量{:i},这个变量会被自动替换为当前节点的opts.data[].i值。
		

		

			opts.target
			[可选]opts.url默认位置。
		

		

			opts.selectleaf=false|true
			[可选]只能选择叶节点。默认值：false。
		

	

	
说明

	
此方法生成可展开的树型HTML选择代码。

	
opts.data结构说明：

	
	[
		{
			i:1,	//节点ID,
			t:'',	//显示文本
			g:'',	//图标路径
			a:'',	//节点URL，这个优先于opts.url。此参数支持自定义变量{:i},这个变量会被自动替换为当前节点的opts.data[].i值。
			u:'',	//用户自定义数据
			e:false|true	//标识节点不可选择
			c:[...],	//子节点,数据结构同本结构
			d:'',	//子节点从网络读取的地址。如果同时设置了c和d，则以c为准。
		},
		...
	]

	
opts.onclick中，可以通过e.target.parentElement.data获取此节点的数据信息。
