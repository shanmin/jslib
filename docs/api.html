$js().add

定义

$js().add(selector) : JSLib

$js().add(HtmlString) : JSLib

$js().add(HtmlElement) : JSLib

参数

selector [可选]指定选择器字符串。
HtmlString [可选]指定的HTML字符串。
HtmlElement [可选]指定HTML节点。

说明

向当前匹配节点中添加新的节点。

示例

HTML 
<div id="a1">
	<div id="a2">a2 text</div>
	<div id="a3">a3 text</div>
</div>
JavaScript 
$js('#a2').add('#a3').addClass('newclass');

当执行$js('#a2')时，此对象会匹配a2节点。再执行add后，此对象会增加a3节点到匹配列表中。

执行后原HTML信息变为 
<div id="a1">
	<div id="a2" class="newclass">a2 text</div>
	<div id="a3" class="newclass">a3 text</div>
</div>

版本

本方法自0.0.0版本开始支持。

$js().appendTo

定义

$js().appendTo(HtmlElements) : JSLib

参数

HtmlElements [必须]指定的节点列表。

说明

当前匹配节点添加到指定的节点下。

当HtmlElements中存在多个节点时，则最终只会添加到第一个节点下。

示例

HTML 
<div id="a2">a2 text</div>
<div id="a1">
	<div id="a3">a3 text</div>
</div>
JavaScript 
$js('#a2').appendTo($js('#a1'));
执行后原HTML信息变为 
<div id="a1">
	<div id="a2">a2 text</div>
	<div id="a3">a3 text</div>
</div>

版本

本方法自0.0.0版本开始支持。

$js().css

定义

$js().css(name) : String

$js().css(name,value) : JSLib

参数

name [必须]css名称。
value [可选]css值。

说明

如果未传递value，则返回对象中第一个节点的css属性值

如果传递value，则设置对象中所有节点的css属性

示例

HTML 
<div id="a1">
	<div id="a2">a2 text</div>
	<div id="a3">a3 text</div>
</div>
JavaScript 
$js('#a2').css('color','#f00');
运行后原HTML变为 
<div id="a1">
	<div id="a2" style="color:#f00;">a2 text</div>
	<div id="a3">a3 text</div>
</div>

版本

本方法自0.0.0版本开始支持。

$js().each

定义

$js().each(func) : JSLib

参数

func [必须]回调函数function(index,element)，参数index为节点的编号，element为节点。

说明

遍历当前对象中的所有节点。

示例

HTML 
<div id="a1">a1 text</div>
<div id="a2">a2 text</div>
<div id="a3">a3 text</div>
JavaScript 
$js('div').each(function(index,element){console.log(index,element.innerHTML)});
输出示例 
0	<div id="a1">a1 text</div>
1	<div id="a2">a2 text</div>
2	<div id="a3">a3 text</div>

版本

本方法自0.0.0版本开始支持。

$js().html

定义

$js().html() : String

$js().html(HtmlString) : JSLib

参数

HtmlString [可选]指定的HTML字符串。

说明

当未传递HtmlString时，将返回第一个匹配节点的HTML文本。

当传递HtmlString时，会使用这个HtmlString字符串内容填充所有匹配的节点。

示例

HTML 
<div id="a1">
	<div id="a2">a2 text</div>
	<div id="a3">a3 text</div>
</div>
JavaScript 
console.log($js('#a1').html());
输出示例 
<div id="a2">a2 text</div>
<div id="a3">a3 text</div>
JavaScript 
$js('#a1').html('<div>abc</div>');
执行后原HTML信息变为 
<div id="a1">
	<div>abc</div>
</div>
